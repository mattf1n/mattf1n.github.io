I’ve been looking for papers that engage with the whether model collapse is a problem for LLMs. This paper looks like it does.

I hadn’t thought about PEFT for compressed models before so this idea caught my attention and I talked to these authors a bit. My understanding is that LoRA trained with a full model doesn’t work so well when the the model gets compressed. So they train another LoRA with a non-linearity to correct the degradation. Their explanation is that the correct update for a compressed model is no longer expressible in low-rank. I’m not convinced that this is the best way, it seems fundamentally inelegant to just add on more parameters, and it seems like there should be an equivalent PEFT method that adds no parameters.
