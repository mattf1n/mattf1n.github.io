Making this site

I have spent a good amount of time thinking about how my site looks. There are a few guiding ideas.

  • No javascript, except for MathJAX.
  • Only HTML and CSS that I edit directly.
  • No external fonts, minimal extra scripts.
  • Make it pretty.
  • Make it responsive.
  • Leave defaults.
  • Don't over-organize.

"Leave defaults" and "make it pretty" conflict from time to time. For instance, adding margins and limiting text width.
